01-JAVA基础类型

2024-06-20  | 作者: 2Dreams

byte

  • 8位，有符号的，以二进制补码表示的整数
  • 取值范围是 [-128,127]
  • 默认值：0

short

  • 16位，有符号的，以二进制补码表示的整数
  • 取值范围$[-32768,32767]$ 即 $[-2^{15},2^{15}-1]$
  • 默认值：0

int

  • 32位、有符号的，以二进制补码表示的整数
  • 取值范围: $[-2^{31},2^{31}-1]$
  • 默认值：0
  • 一般地，整形变量默认为int类型

long

  • 64位，有符号的以二进制补码表示的整数
  • 取值范围 $[-2^{63},2^{63}-1]$
  • 默认值：0L

float

  • float数据类型是单精度、32位、符合IEEE 754标准的浮点数
  • 默认值： 0.0f

double

  • float数据类型是单精度、64位、符合IEEE 754标准的浮点数
  • 浮点数默认类型位double类型
  • 默认值：0.0d

boolean

  • boolean 数据类型表示一位的信息
  • 只有两个取值： true和false
  • 默认值： false

char

  • char类型是一个单一的16位的Unicode字符
  • 取值范围：[\u0000,\uffff] 即 $[0,65535]$
  • char 数据类型可以存储任何字符
